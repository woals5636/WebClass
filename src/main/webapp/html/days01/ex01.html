1. 인터넷(Internet)
    • 전 세계를 연결하는 국제 정보 통신망
    • ARPA(Advanced research projects Agency)가 1969년에 웹의 모태가 되는 ARPANET 개발.
    • 군사용과 민간용 구분
    • 민간용으로 사용되던 네트워크가 현재의 인터넷이 되었다.
  2. 웹(web)
    1. 인터넷에서 작동되는 서비스
    2. 1991년 등장
    3. 인터넷에 연결된 컴퓨터를 통해서 사람들이 정보(영상,문자 등등)을 공유하는 공간
    4. 1989년 유럽 공동 원자핵 연구소(CERN)의 "팀버너스 리"가 하이퍼 링크를 포함하는 문서의 개념을 제안 -> 1991년 "월드 와이드 웹(WWW)"을 개발해서 배포
    5. W3C(World Wide Web Consortium)
      1. 1994년 10월 "팀버너스 리"에 설립
      2. 국제 웹 표준화 단체(기구)
      3. html, css3, xml 등등 표준화, 강제성 X
      4. WWW의 표준안 제작과 새로운 표준안 제안 기술의 공유를 통해서 WWW의 기술적, 사회적 확장을 위해 구성된 세계적 단체
  3. 웹 브라우저
    1. 인터넷에서 정보를 공유(검색)하는 응용 프로그램
    2. 최초의 웹브라우저 - 모자이크(1993년)- > "넷스케이프" 발표 넷스케이프의 개발 코드명 : 모질라(Mozila) ㄴ 모자이크 웹브라우저를 파괴하겠다.
    3. W3C의 표준을 무시하고 경쟁 - (1차 웹 브라우저 전쟁 1994~1998) MS사 - IE ( PC에 강제 설치 ) 승리. 넷스케이프사 - 넷스케이프 웹 브라우저 사이에 발생하는 기술적인 전쟁..
    4. 플러그인
      • 웹 브라우저와 연동되는 특정 프로그램. 사용자가 PC에 추가로 설치하여 웹 브라우저의 기능 을 확장하는 방법
      • 웹의 발전 속도를 웹브라우저가 따라가지 못해서 기업 개발.
      • 대표적인 플러그인 : 어도비 플래시, MS사 액티브X 등등
    5. 웹 1.0 웹 2.0 ( 사용자 새로운 켄트츠를 창조 ) 웹 3.0 ( 인터넷 엄청난 양의 정보 -> 필요한 정보/지식만을 추출해서 보여주는 맞춤형 시대 )
  4. 플러그인의 문제 : 결제 - 액티브X , (점점 플러그인 설치-> 웹 사이트 무거워짐.) HTML 표준 제안서 제출 -> MS사(IE) 거절.
  5. MS사(IE) 제외한 그 외의 웹 브라우저 제공 업체 WHATWG 설립. WHATWG( Web Hypertext Application Technology Woking Group ) 2004년 6월 html5 표준을 제정 새로운 웹 표준(Web Application 1.0)을 작성 W3C - Web Application 1.0 표준으로 채택... ==> html5 웹 표준 변경.
  6. 2차 웹 브라우저 전쟁 시작 MS사(IE) <-> 그 외 모든 웹브라우저( WHATWG 승리 ) 최신 웹 표준지원 X
  7. html 역사
