js 코드는 <script></script> 태그 안에 코딩한다. <script type="text/javascript"></script> type="text/javascript" 속성은 필수 아님 두 군데( body , head ) 에 추가할 수 있다. body - <script></script> head - <script></script> <script></script> 요소를 </body> 위에 배치하면 스크립트 해석으로 인해 표시 속도가 향상된다. 여러 웹 페이지에서 공통적으로 사용되는 js 코딩이 있다면 외부 js 파일로 참조할 수 있다. html 코딩 + js 코딩 분리 ㄴ 가독성 향상 , 유지/보수 용이 ㄴ 외부 js 파일 캐싱되어서 페이지의 로딩 속도가 향상 된다.
